# Rdd

This gem is written to solve the following problem: https://gist.github.com/coryodaniel/61a2ebf7fe5086b69f36

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'rdd'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rdd
